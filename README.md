# Project excelling at Excel

A self-structured project to build Excel proficiency.

## About

Excel is the workhorse of most finance and accounting work, so instead of picking it up piecemeal I decided to learn it properly and on purpose.

The project works through five modules: core mechanics and keyboard speed, the functions library (IF, SUMIF, INDEX/MATCH, XLOOKUP), data and analysis tools, financial modeling, and accounting-specific work like depreciation schedules and reconciliations.
